# AEGIS — Agent Runtime Platform Roadmap

**Vision:** The infrastructure layer for AI agents. Deploy, orchestrate, and monetize AI agents at scale.

**Position:** Between AWS Bedrock (too low-level) and OpenClaw (single-user tool). AEGIS is the managed platform where teams deploy production agent workflows.

**Business Model:**

- Free: 3 agents, 100 runs/month
- Pro: Unlimited agents, 5K runs/month — $49/mo
- Team: Multi-user, shared agents, analytics — $199/mo
- Enterprise: Self-hosted, SOC2, SLA — $999+/mo

## Phase 1: Core Platform API (Weeks 1-3)

### Goal: Developers can create and run agents via API

#### 1.1 Agent API (REST)

```
POST   /v1/agents                    — Create agent
GET    /v1/agents                    — List agents
GET    /v1/agents/{id}               — Get agent details
PUT    /v1/agents/{id}               — Update agent
DELETE /v1/agents/{id}               — Delete agent
POST   /v1/agents/{id}/run           — Run agent with a task (sync)
POST   /v1/agents/{id}/run/async     — Run agent (async, returns task_id)
GET    /v1/tasks/{id}                — Get task status/result
GET    /v1/tasks/{id}/stream         — SSE stream for progress
POST   /v1/agents/{id}/chat          — Conversational (stateful)
```

#### 1.2 Authentication & Security

- API keys: `sk-aegis-xxxx` (hashed in DB, never stored plain)
- JWT for web dashboard sessions
- Rate limiting per API key (token bucket)
- CORS with allowlist
- Request signing for webhooks
- Input validation on all endpoints
- Audit log for every API call (we already have this)
- Sandbox isolation (gVisor — already have this)
- Multi-tenant data isolation (tenant_id on every table)

#### 1.3 Agent Definition Format

```yaml
# Standard agent definition (evolution of our .md format)
name: financial-analyst
version: 1.0.0
description: Analyzes financial data and produces reports
model: claude-sonnet # or: gpt-5, gemini-2.5-pro, custom
tools:
  - web_search
  - code_execute
  - db_query
  - file_read
  - file_write
autonomy: green # green/yellow/red
max_iterations: 20
max_tokens: 8000
system_prompt: |
  You are a financial analyst...
skills:
  - quant-analytics
  - crypto-dashboard
metadata:
  author: "team-alpha"
  tags: ["finance", "analytics"]
  public: false
```

#### 1.4 Database Schema Changes

```sql
-- Multi-tenant foundation
CREATE TABLE tenants (
    id UUID PRIMARY KEY,
    name TEXT NOT NULL,
    plan TEXT DEFAULT 'free',
    created_at TIMESTAMPTZ DEFAULT now()
);

CREATE TABLE api_keys (
    id UUID PRIMARY KEY,
    tenant_id UUID REFERENCES tenants(id),
    key_hash TEXT NOT NULL,        -- bcrypt hash of sk-aegis-xxx
    key_prefix TEXT NOT NULL,      -- first 8 chars for display
    name TEXT,
    permissions JSONB DEFAULT '["agents:read","agents:run"]',
    rate_limit_rpm INTEGER DEFAULT 60,
    created_at TIMESTAMPTZ DEFAULT now(),
    last_used_at TIMESTAMPTZ
);

CREATE TABLE agents (
    id UUID PRIMARY KEY,
    tenant_id UUID REFERENCES tenants(id),
    name TEXT NOT NULL,
    version TEXT DEFAULT '1.0.0',
    definition JSONB NOT NULL,     -- full agent config
    public BOOLEAN DEFAULT false,
    runs_count INTEGER DEFAULT 0,
    created_at TIMESTAMPTZ DEFAULT now(),
    updated_at TIMESTAMPTZ DEFAULT now()
);

-- Add tenant_id to existing tasks table
ALTER TABLE tasks ADD COLUMN tenant_id UUID REFERENCES tenants(id);
ALTER TABLE tasks ADD COLUMN agent_id UUID REFERENCES agents(id);
```

## Phase 2: Developer Experience (Weeks 4-6)

### Goal: Developers love using AEGIS

#### 2.1 Python SDK

```python
pip install aegis-sdk

from aegis import Aegis, Agent

client = Aegis(api_key="sk-aegis-xxx")

# Create an agent
agent = client.agents.create(
    name="code-reviewer",
    model="claude-sonnet",
    tools=["web_search", "code_execute"],
    instructions="You review code for bugs and security issues..."
)

# Run synchronously
result = agent.run("Review this Python file for security issues: ...")
print(result.output)
print(result.cost)
print(result.duration)

# Run async with streaming
async for event in agent.stream("Analyze competitor pricing for Stripe"):
    if event.type == "narration":
        print(f"  → {event.data}")
    elif event.type == "complete":
        print(f"\nResult: {event.result}")

# Conversational (maintains context)
chat = agent.chat()
chat.send("What's the current BTC price?")
chat.send("Compare it to last week")  # remembers context
```

#### 2.2 JavaScript/TypeScript SDK

```typescript
import { Aegis } from "@aegis/sdk";

const client = new Aegis({ apiKey: "sk-aegis-xxx" });

const result = await client.agents.run("financial-analyst", {
  task: "Generate Q3 revenue report",
  webhookUrl: "https://myapp.com/webhook/aegis",
});
```

#### 2.3 CLI Tool

```bash
# Install
pip install aegis-cli

# Auth
aegis login

# Create agent from YAML
aegis agent create --file agent.yaml

# Run
aegis run financial-analyst "Analyze Shopify Q3 revenue"

# List
aegis agents list
aegis tasks list --status running
```

- Select model, tools, skills from dropdowns

#### 3.3 Security for Web UI

- JWT sessions with refresh tokens
- OAuth (Google, GitHub login)
- RBAC: admin, developer, viewer roles
- Session timeout (24h)
- 2FA for enterprise plans
- All API calls go through the same auth middleware
- CSP headers, XSS protection
- No secrets ever sent to frontend

## Security Architecture

```
┌─────────────────────────────────────────────┐
│               Security Layers               │
├─────────────────────────────────────────────┤
│ L1: Network    │ HTTPS, WAF, DDoS protect  │
│ L2: Auth       │ API keys, JWT, OAuth, SSO │
│ L3: Authz      │ RBAC, tenant isolation    │
│ L4: Input      │ Validation, sanitization  │
│ L5: Execution  │ gVisor sandbox, no-net    │
│ L6: Output     │ PII masking, filtering    │
│ L7: Audit      │ Hash-chain log, immutable │
│ L8: Data       │ Encryption at rest/transit│
└─────────────────────────────────────────────┘
```
